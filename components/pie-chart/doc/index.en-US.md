| Property | Description | Type | Default |

### [vts-radio]

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |

| vtsAutoFocus | Set autofocus on mounted | `boolean` | `false`
| vtsValue | Value of option | `any` |
